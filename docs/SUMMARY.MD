### Stage 4: Formalization via Free Energy Principle

The vague term "self-reference" was replaced with a precise physical formalism using **Markov blankets** (from Karl Friston's Free Energy Principle).

**State partition:**
- **I** = Internal states (the system's inner workings)
- **E** = External states (the environment)
- **S** = Sensory states (how E influences I)
- **A** = Active states (how I influences E)
- **Markov blanket** = S вҲӘ A

**The dimensionality crisis:**
For a complete self-model, I must encode a generative model of I, S, and A. But I is constituted by a finite number of degrees of freedom. A complete 1:1 self-model requires more degrees of freedom than the system possesses. This is a **Turing limit** (capacity constraint), not a GГ¶del limit (logical paradox).

**The infinite regress:**
1. System updates self-model q to reflect current state p
2. The act of updating changes p (heat dissipation perturbs the substrate)
3. System must immediately compute another update
4. Repeat вҖ” each update creates the need for the next update

Where:

**Step 1 вҖ” write this and make sure it runs without errors:**

```python
import numpy as np

def self_model_dynamics(state, t, eta, alpha):

```

**Step 2 вҖ” add the update current (the first equation):**
```python
    J = -eta * np.log((q * (1 - p)) / (p * (1 - q) + 1e-12))
```
The `1e-12` is a numerical guard against division by zero when q вүҲ p.

**Step 3 вҖ” add the physical state perturbation (the second equation):**
```python
    dq_dt = J
    dp_dt = alpha * np.abs(J)
    return [dq_dt, dp_dt]
```

**Step 4 вҖ” test it with scipy's ODE integrator:**
```python
from scipy.integrate import odeint

# initial conditions: model slightly wrong, physical state at 0.6

# static case: alpha=0 (no regress, should converge cleanly)
```

**What you should see:** the static case (`alpha=0`) converges вҖ” q catches up to p and stays there. The dynamic case (`alpha>0`) never converges вҖ” p keeps running away from q. That divergence is the regress. That is the entire theory visible in two lines of output.
