# Thermodynamic AI: Self-Transcending Neural Architecture

This project implements a "Thermodynamic AI" system that overcomes the "Architectural Trap" (local minima in model structure) by using deterministic chaos to drive topological mutations.

## The Core Concept
Standard neural networks optimize parameters (weights) within a fixed architecture. This system optimizes the architecture itself.
When the system hits a "Thermodynamic Barrier" (loss plateau), it injects a "Chaotic Push" derived from a Lorenz Attractor to:
1.  **Resize the Brain**: Dynamically add/remove neurons (Topological Mutation).
2.  **Shift Neurochemistry**: Swap activation functions (e.g., ReLU -> Tanh) during high-chaos states (Chemical Phase Shift).
3.  **Preserve Memory**: Transfer weights from the old structure to the new one (Neuroplasticity).

## Project Structure
*   `test.py`: The main engine. Contains the `LorenzGenerator`, `ChaosInjector`, `ArchitectureMonitor`, and the training loop.
*   `visualize.py`: A tool to plot the "Thermodynamic Staircase" (Loss vs. Architecture Size).
*   `thermodynamic_training_log.csv`: The log file generated by `test.py`.
*   `AGENTS.md`: Description of the microservices/agents in the system.

## How to Run
1.  **Train the Agent**:
    ```bash
    python test.py
    ```
    Watch the logs for `>>> CHAOS INJECTION` and `>>> MEMORY PRESERVED`.

2.  **Visualize the Evolution**:
    ```bash
    python visualize.py
    ```
    This will generate `thermodynamic_evolution.png`.
